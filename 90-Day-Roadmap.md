# 📅 Complete 90-Day Java Developer Roadmap

> A day-by-day breakdown of your journey from beginner to job-ready Java Developer.

---

## 🟢 Phase 1: Programming Fundamentals (Days 1–4)

> **Goal:** Build a solid foundation in programming concepts using Java syntax.

| Day | Topics | Key Concepts |
|-----|--------|-------------|
| Day 1 | Introduction to Java, JDK Setup, First Program | What is Java, JVM/JDK/JRE, "Hello World", compilation |
| Day 2 | Variables, Data Types, Type Casting | Primitive types, reference types, type conversion |
| Day 3 | Operators and Expressions | Arithmetic, relational, logical, bitwise, ternary |
| Day 4 | Input/Output, Control Flow (if-else, switch) | Scanner class, conditional statements, nested conditions |

### 🎯 Phase 1 Milestone
- [ ] Can write and compile Java programs from scratch
- [ ] Understand Java's type system
- [ ] Can take user input and make decisions in code

---

## 🟢 Phase 2: Core Java (Days 5–9)

> **Goal:** Master loops, arrays, strings, and methods — the building blocks of all programs.

| Day | Topics | Key Concepts |
|-----|--------|-------------|
| Day 5 | Loops (for, while, do-while) | Loop types, nested loops, loop control (break, continue) |
| Day 6 | Arrays (1D and 2D) | Array declaration, traversal, searching, sorting |
| Day 7 | Strings & StringBuilder | String methods, immutability, StringBuilder, StringBuffer |
| Day 8 | Methods/Functions | Method declaration, parameters, return types, overloading |
| Day 9 | Recursion & Practice Day | Recursive thinking, base case, stack overflow, practice |

### 🎯 Phase 2 Milestone
- [ ] Can solve problems using loops and arrays
- [ ] Understand string manipulation thoroughly
- [ ] Can write reusable methods and recursive solutions
