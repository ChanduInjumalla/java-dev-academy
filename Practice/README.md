# 💻 Practice Directory

Welcome to the Practice directory! This is your sandbox.

## How to use this directory:

1. Use this space to write code while following along with the daily topics and examples.
2. Store your solutions for the "10 Coding Questions" provided in the Extra Practice section of each daily plan.
3. Experiment with different Java features here without worrying about breaking larger projects.

**Example Structure:**
```
Practice/
├── Phase-02-Core-Java/
│   ├── LoopsPractice.java
│   ├── ArrayOperations.java
│   └── StringManipulation.java
├── Phase-03-OOP/
│   ├── ClassDemo.java
│   └── InheritancePractice.java
...
```

**Tip:** Organize your practice files logically so you can refer back to them when preparing for interviews!
