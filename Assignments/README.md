# 📝 Assignments Directory

Welcome to the Assignments directory! This is where you will store the solutions to the assignments given at the end of each daily plan.

## How to use this directory:

1. Create a folder for each phase or day. For example: `Phase-01/Day-01/` or simply `Day-01/`.
2. Write your Java code to solve the assignments.
3. Add comments to your code explaining your logic.
4. If an assignment requires a text answer, create a `.md` or `.txt` file.

**Example Structure:**
```
Assignments/
├── Day-01/
│   ├── HelloWorld.java
│   └── ProfilePrinter.java
├── Day-02/
│   ├── Calculator.java
│   └── TemperatureConverter.java
...
```

Remember: **Practice makes perfect.** Don't skip the assignments!
