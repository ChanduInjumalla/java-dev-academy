## 📖 Recommended Books

### For Beginners
| Book | Author | Best For |
|------|--------|---------|
| Head First Java | Kathy Sierra | Absolute beginners, visual learners |
| Java: The Complete Reference | Herbert Schildt | Comprehensive reference |
| Java in a Nutshell | Benjamin Evans | Quick reference guide |

### For Intermediate
| Book | Author | Best For |
|------|--------|---------|
| Effective Java | Joshua Bloch | Best practices, must-read! |
