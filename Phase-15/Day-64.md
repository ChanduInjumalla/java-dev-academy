# 📅 Day 64: REST Fundamentals

## 🎯 Goal of the Day
Begin **Phase 15: REST APIs**. Understand RESTful architecture principles and HTTP fundamentals.

## 📚 Topics Covered
1. What is REST? (Representational State Transfer)
2. REST Principles (Stateless, Client-Server, Uniform Interface)
3. HTTP Methods (GET, POST, PUT, PATCH, DELETE)
4. HTTP Status Codes (200, 201, 204, 400, 401, 403, 404, 500)
5. JSON format
6. API Design Best Practices
7. Resource naming conventions

## Key Concepts
| Method | Purpose | Idempotent | Example |
|--------|---------|-----------|---------|
| GET | Read | Yes | GET /api/users |
| POST | Create | No | POST /api/users |
| PUT | Update (full) | Yes | PUT /api/users/1 |
| PATCH | Update (partial) | No | PATCH /api/users/1 |
| DELETE | Delete | Yes | DELETE /api/users/1 |
